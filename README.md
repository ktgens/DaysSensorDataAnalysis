Программа представляет собой датчик, собирающий показания температуры. Датчик считывает D дней (до 30) по H измерений (до 24) температуры в диапазоне от -50 до 50°C.

Для хранения данных используются двумерные динамические массивы. Программа считает общую среднюю температуру и выводит дни, где средняя температура больше общей средней.

Для работы с массивами используются указатели и арифметика указателей.

В конце программы производится освобождение памяти и обнуление указателей.

### Пример вывода:
```
===Анализ сенсора===
Введите количество дней (до 30): 3
Введите количество измерений в дне (до 24): 4
День 1: 8 9 13 12
День 2: 3 4 7 8
День 3: 20 23 30 25

Общая средняя температура: 13.5
Теплые дни:
День 3 (среднее = 24.5)
```
